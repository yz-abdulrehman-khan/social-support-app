# Screenshots Directory

Place your screenshot images in this directory with the following naming convention:

## English Screenshots
- `en-landing.png` - Landing page
- `en-step1.png` - Personal Information form
- `en-step2.png` - Family & Financial Details form
- `en-step3.png` - Situation Description form
- `en-step4.png` - Review & Confirm page
- `en-ai-modal-empty.png` - AI Assistant modal (empty state)
- `en-ai-modal-generated.png` - AI Assistant modal (with generated text)
- `en-ai-modal-loading.png` - AI Assistant modal (loading state)
- `en-success.png` - Success confirmation page
- `en-validation.png` - Form with validation errors
